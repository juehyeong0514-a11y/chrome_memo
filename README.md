# Web Annotation Floating Pen

웹 페이지 위에 바로 필기하고, 형광펜 표시, 지우개, 텍스트 메모, 화면 캡처를 사용할 수 있는 Chrome Manifest V3 확장 프로그램입니다.

확장을 켜면 기본적으로 바로 펜 그리기 모드로 시작합니다. 별도로 펜을 다시 선택하지 않아도 곧바로 필기할 수 있습니다.

## 주요 기능

- Chrome Extension Manifest V3 기반
- `http`, `https`, `file` 페이지 지원
- 확장 프로그램 팝업에서 전체 ON/OFF
- 확장 실행 시 기본 `펜 + 그리기 모드`로 즉시 진입
- 플로팅 원형 버튼과 펼침 도구막대
- 펜, 형광펜, 지우개, 텍스트, 선택/이동, 탐색 모드
- 펜 종류별 색상, 굵기, 투명도, 압력 감도, 둥글기 설정
- 그리기 모드에서도 마우스 휠로 페이지 스크롤 가능
- 일반 문서 스크롤과 내부 스크롤 컨테이너 모두 대응
- 내부 스크롤 영역에서 필기와 텍스트가 화면 이동에 맞춰 함께 이동
- URL별 필기와 텍스트 자동 저장 및 복원
- 선택한 stroke 또는 텍스트 이동, undo/redo 기록
- 전체 화면 캡처와 선택 영역 캡처
- PNG 원본 품질을 유지하는 캡처 미리보기, 저장, 클립보드 복사
- 도구막대 드래그 이동, 크기 조절, 가로/세로 방향 설정
- 도구막대 위치, 크기, 방향, 언어 설정 자동 복원
- Shadow DOM 기반 UI로 페이지 CSS 충돌 방지
- `devicePixelRatio` 대응 Canvas 렌더링
- `requestAnimationFrame` 기반 렌더링 최적화

## 사용 흐름

1. 확장 프로그램을 설치하고 페이지를 엽니다.
2. 확장이 ON 상태라면 도구막대가 표시되고 기본 펜 모드가 활성화됩니다.
3. 페이지 위에서 바로 필기합니다.
4. 그리기 모드에서도 마우스 휠을 굴리면 페이지가 스크롤됩니다.
5. 필요한 경우 도구막대에서 형광펜, 지우개, 텍스트, 선택/이동, 탐색 모드로 전환합니다.

`Esc`를 누르면 탐색 모드로 전환됩니다. 다시 필기하려면 펜 버튼을 누르면 됩니다.

## 도구막대

도구막대에는 실행 취소, 다시 실행, 펜, 형광펜, 지우개, 텍스트, 캡처, 선택/이동, 탐색 모드, 숨김 기능이 있습니다.

- 원형 플로팅 버튼을 누르면 도구막대가 펼쳐집니다.
- 원형 버튼에는 현재 펜 색상과 굵기 미리보기가 표시됩니다.
- 원형 버튼을 드래그해서 위치를 옮길 수 있습니다.
- 펼친 도구막대의 드래그 핸들을 잡고 위치를 옮길 수 있습니다.
- 오른쪽 아래 모서리를 드래그해서 도구막대 크기를 조절할 수 있습니다.
- 팝업 환경설정에서 도구막대 크기, 가로/세로 방향, 언어를 바꿀 수 있습니다.
- 위치, 크기, 방향은 `chrome.storage.local`에 저장되어 다음 실행 때 복원됩니다.

## 필기와 스크롤

그리기 모드에서도 마우스 휠은 페이지 이동으로 동작합니다.

- 일반 문서 스크롤 페이지에서는 `window` 스크롤을 따라 필기가 함께 이동합니다.
- 페이지 내부의 별도 스크롤 영역에서는 마우스 아래 실제 스크롤 컨테이너를 찾아 이동합니다.
- 내부 스크롤 컨테이너에서 새로 그린 필기와 텍스트는 해당 컨테이너의 스크롤 위치를 기준으로 저장됩니다.
- 내부 스크롤이 움직이면 캔버스를 다시 렌더링해서 필기가 화면 위치에 맞게 따라가도록 처리합니다.

기존에 저장된 일부 필기는 예전 좌표 방식으로 저장되어 특정 내부 스크롤 페이지에서 위치가 다르게 보일 수 있습니다. 새로 작성하는 필기는 현재 방식으로 저장됩니다.

## 도구

- 펜: 볼펜, 만년필, 붓 타입을 지원합니다.
- 형광펜: 현재 선택한 펜 색상과 투명도 기반으로 표시합니다.
- 지우개: 크기 설정을 지원하며 stroke 단위로 지웁니다.
- 텍스트: 페이지 위에 텍스트 박스를 만들고 이동, 크기 조절, 삭제할 수 있습니다.
- 선택/이동: stroke 하나 또는 텍스트 하나를 선택해서 드래그 이동할 수 있습니다.
- 탐색 모드: 필기는 보이지만 페이지 링크와 버튼을 조작할 수 있습니다.
- 캡처: 전체 화면 캡처와 선택 영역 캡처를 지원합니다.

## 선택/이동

- 선택/이동 버튼을 누르면 선택 모드가 됩니다.
- stroke의 가까운 위치를 클릭하면 해당 stroke가 선택됩니다.
- 텍스트 박스 하나를 선택하고 드래그해서 이동할 수 있습니다.
- 이동 작업은 `undo`/`redo` 기록에 포함됩니다.
- 다중 선택, 그룹 이동, 속성 일괄 변경은 아직 지원하지 않습니다.

## 커서 미리보기

- 캔버스 영역에서만 커서 미리보기가 표시됩니다.
- 도구막대나 팝업 위에서는 커서 미리보기가 숨겨집니다.
- 펜과 형광펜은 실제 굵기를 원형 커서로 표시합니다.
- 지우개는 실제 지우는 범위를 원형으로 표시합니다.
- 흰색과 검정색 이중 테두리를 사용해 밝은 배경과 어두운 배경 모두에서 보이도록 합니다.

## 캡처

- Chrome `tabs.captureVisibleTab`으로 PNG 형식 캡처를 생성합니다.
- 미리보기는 캡처 data URL을 그대로 `<img>`에 넣어 표시합니다.
- 미리보기 표시 과정에서 JPEG 변환이나 Canvas 재샘플링을 하지 않습니다.
- 선택 영역 캡처는 원본 캡처 이미지에서 선택 영역을 비율대로 잘라 PNG Blob으로 생성합니다.
- 캡처 결과는 저장하거나 클립보드에 복사할 수 있습니다.

## 성능

- 그리기 중 반복되는 Canvas 렌더링은 `requestAnimationFrame`으로 묶어 처리합니다.
- 여러 `pointermove` 이벤트가 들어와도 한 프레임에 한 번만 렌더링합니다.
- 지우개 이동 중 여러 샘플을 처리해도 렌더링은 프레임 단위로 제한합니다.
- undo, redo, 전체 지우기, 저장 복원처럼 즉시 반영이 필요한 작업은 바로 렌더링합니다.

## 저장 방식

필기 데이터는 URL별로 `chrome.storage.local`에 자동 저장됩니다.

저장되는 주요 값:

- `settings.globalEnabled`
- `settings.uiSettings.toolbarScale`
- `settings.uiSettings.toolbarOrientation`
- `settings.uiSettings.language`
- `toolbarPosition`
- `wae:v1:page:<origin><path><search>`
- `penSettings`
- `selectedPenType`
- `recentColors`
- `customColors`
- `eraserSettings`
- `textSettings`

페이지별 주석 저장 값:

- `strokes`
- `textItems`
- `updatedAt`
- `version`

## 팝업

확장 프로그램 아이콘 팝업에서 다음 작업을 할 수 있습니다.

- 확장 프로그램 ON/OFF
- 도구막대 크기 조절
- 도구막대 가로/세로 방향 선택
- 언어 선택
- 현재 페이지 전체 지우기
- 도구막대 위치 초기화
- 현재 페이지 지원 여부 확인

OFF로 전환하면 현재 페이지의 메모리와 저장소에서 주석 데이터가 지워집니다.

## 단축키

- `Esc`: 탐색 모드로 전환
- `Ctrl+Z`: 실행 취소
- `Ctrl+Shift+Z`: 다시 실행
- `Ctrl+Y`: 다시 실행
- `Shift` 누른 채 필기: 직선 보정 모드

입력창, textarea, select, contenteditable 요소에 포커스가 있을 때는 페이지 기본 입력을 방해하지 않습니다.

## 설치

1. Chrome 주소창에 `chrome://extensions`를 입력합니다.
2. 오른쪽 위의 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 프로젝트 폴더를 선택합니다.

```text
c:\Users\User\Desktop\chorm_meno
```

## 제한 사항

- Chrome 내부 페이지, Chrome Web Store, 확장 프로그램 관리 페이지에서는 content script가 차단됩니다.
- Chrome 내장 PDF 뷰어는 현재 지원하지 않습니다.
- iframe 내부 필기는 브라우저 보안 정책과 페이지 구조에 따라 제한될 수 있습니다.
- `file://` 주소에서 사용하려면 Chrome 확장 프로그램 상세 정보에서 파일 URL 접근 권한을 켜야 합니다.
- 기존 좌표 방식으로 저장된 일부 주석은 내부 스크롤 페이지에서 새 좌표 방식과 다르게 보일 수 있습니다.
